Latest Projects

Saliency based Image Segmentation

In this work, we present a novel approach for refining segmentation using saliency map. To achieve this, we first develop a new saliency detection method based on cues at various levels: local, global and rarity. Further, to increase the discriminability of the proposed saliency map, we apply a non-linear scale space preprocessor unit to the image. Finally the saliency map is used as initialization for segmentation. This technique is applied successively to perform multi-object segmentation.

Characterizing Objects

In this respect, we have developed a novel set of features using a combination of KAZE and SIFT keypoints termed as SIKA features. We have shown the efficacy of this feature set for object classification. SIKA features were designed to effectively characterize the objects. This is achieved by SIKA because these features serve as a good combination for boundary (KAZE) and saliency (SIFT) information. We have also shown that SIFT and KAZE are complementary features. Utilizing SIKA features, we develop a salient keypoint selection strategy based on the discriminability, detectability and repeatability of keypoints and show their effectiveness in object representation

Saliency and object feature driven segmentation

As, saliency map alone cannot always give the best segmentation because at times some background regions can be marked as salient. Also, the segmentation should closely fit the object contours to get neat segmentation. To overcome these issues, the technique is based on saliency and image features (Scale Invariant Feature Transform (SIFT), KAZE etc.) for characterization of objects and subsequent segmentation.

Ordered salient object proposals

We aim at demonstrating better performance (high recall and precision rates) utilizing very few number of object proposals. Thus, we are able to provide a tight bound on the number of object proposals. We rank the key objects in relative order of salience using the generated box proposals leading to lower complexity segmentation technique.
